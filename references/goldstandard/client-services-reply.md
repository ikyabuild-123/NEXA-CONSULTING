---

Web scraping — we build systems that pull structured data from the web at scale (candidate profiles, company signals, market intel) and land it clean in whatever you already use: a sheet, your ATS, a database. Every run is reviewable before it touches live data.

## What makes this work

- **Subject line:** "Your question about..." — human, not transactional
- **Opening:** confirms the ask immediately, no fluff
- **Structure:** each service defined with a single em-dash sentence, then what it means practically
- **Client-specific paragraph:** names Kontrast, names their use cases — shows homework was done
- **Ask:** specific (20 minutes, this week), low commitment, adds value ("share examples")
- **Sign-off:** "Talk soon" — warm. Lowercase "founder, nexa consulting" — personal, not branded
